HTML

-> HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page.

TAGS

  1. < head > ___ < /head >

    • This is the tag which tells what the website do and some info about website

    • It represents the document's header which keep other HTML Tags like < title > --- < /title > &
      < link > --- < /link >.

  2. < h1 > ___ < /h1 >

    • this is the Heading tag .

    • It start from h1 and ends at h6

    • h1 is the bigest heading and h6 is the lowest heading

  3. < p > ___ < /p >

    • this is the paragraph tag .

    • if you want to write paragraph then you use this tag

    • Each paragraph of text should go between an opwning < p > & a closing < /p > tag .

HTML FORMATTING

  1. BOLD TEXT

    • you can change the text to bold by using tag < b > --- < /b > .

  2. ITALIC TEXT

    • you can change the text to Italic by using tag < i > --- < /i > .

  3. UNDERLINE TEXT

    • you can change the text to bold by using tag < u > --- < /u > .

  4. STRIKE TEXT

    • you can change the text to bold by using tag < STRIKE > --- < /STRIKE > .

  5. SUPERSCRIPT

    • you can SUPERSCRIPT the words by using tag < sup > -< /sup > .

    • Such as i use 3 < sup> 2 < /sup> = 9 :-
      32 = 9
